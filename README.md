# Now on the [Chrome Web Store](https://chromewebstore.google.com/detail/ezretime/jicnjjcmeekhhjgnlpkobdgakpmngjmn)!

ezRetime is a quick speedrun retime tool that works on Twitch, YouTube, and more! It allows you to go frame-by-frame and generate a mod message with info about those frames. It also supports multiple splits.
